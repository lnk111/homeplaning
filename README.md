# 홈플래닝 (HOMEPLANING)

> 2030 사회초년생의 돈 관리·자산형성·내집마련을 돕는 무료 계산기 플랫폼
> **homeplaning.kr** · 운영: Design For

값만 넣으면 내 상황에 맞는 숫자가 바로 나오는 계산기 5종 + AI가 매일 반영하는 최신 정책.

## 계산기 5종

| 계산기 | 하는 일 |
|--------|--------|
| 🏠 내집마련 | 매매가·소득·지역 → 대출 한도(LTV·DSR·절대캡 최솟값)·필요현금·월상환·정책대출 비교 |
| 📈 복리 | 매월 적립 시 자산 성장 시뮬레이션 (단리 vs 복리) |
| 🎯 자산목표 | 목표 금액에서 역산 → 필요 월 저축액 / 도달 기간 |
| 💰 월급배분 | 실수령액을 50·30·20 등 원칙으로 자동 배분 |
| 🏦 적금·배당 | 청년도약계좌 만기 수령액, 배당 재투자 수익 |

## 기술 스택

- **프론트엔드**: 바닐라 HTML/CSS/JS (프레임워크 없음) — SEO 최적화
- **폰트**: Pretendard (jsDelivr CDN)
- **호스팅**: GitHub Pages + 커스텀 도메인 (HTTPS)
- **AI 정책 파이프라인**: GitHub Actions 매일 크론 → Claude(`claude-opus-4-8`) 웹검색 → `data/policy.json` 자동 갱신·커밋

## 특징

- **정확성**: 2026년 정책(6·27 대책, 스트레스 DSR, 정책대출 금리, 지역 3단계, 신생아 특례) 반영
- **최신성**: 부동산·금융 정책을 AI가 매일 확인해 계산 기준에 반영
- **정직함**: 모든 결과에 "참고용 추정치" 명시, 전문가 상담 권유
- **무료·무가입**: 입력값은 브라우저 내에서만 처리(서버 전송 없음)

## 비즈니스 모델

계산기는 대출·집 마련을 진지하게 고민하는 고관여 방문자를 모읍니다.

1. **대출상담 제휴 리드** (contact 페이지) — 계산 결과 기반 실수요 연결
2. **Design For 인테리어 유입** — 집 구매 → 리모델링 수요 자체 전환
3. **금융상품 제휴** — 청년적금·청약·카드
4. **Google AdSense** — 트래픽 기반 광고 수익(베이스라인)

## 개발

```bash
python -m http.server 8000   # 로컬 미리보기 (http://localhost:8000)
npm run update-policy        # AI 정책 갱신 수동 실행 (ANTHROPIC_API_KEY 필요)
```
